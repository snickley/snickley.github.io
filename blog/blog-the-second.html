Margin vs border vs padding?

Sprint 2 exercise

30 June 2022

What's a margin?

A margin is the space around the outside of an object. Margin is set as a number of pixels, and can be applied to each edge of the object. The default value is 0px. You can apply negative values, meaning that the space around an object would overlap the space around the object next to it.

What's padding?

Padding is the space between the outer edge of the object, and the content held inside the object.
Padding is set as a number of pixels, and can be applied to each edge of the object. The default value is 0px. You can't apply negative values - that would mean the object contents appear outside the box!

What's a border?

A border is the outer edge of the object - it creates the line between the margin and padding. You set the width, style and colour to create a border. The default values are:

width = medium
stlye = none
colour = same colour as the text inside the object

To have a border show up, it needs a width of more than 0 and a style that is not none.

How do you pick which one to use?

Depends what you're trying to achieve! Padding is inside the object, so increased padding will make your object larger - it'll increase the amount of space incorporated in your background colour or clickable region. Margin is outside the object so it won't make the object larger. A border is used when you need delineation between the object and the space around it.

How do you use them?

The margin, padding and border properties are all set within css. Each property can be set to different values for the top, right, bottom, and left of the object.

<!DOCTYPE html>
<html>
<head>
<style>
div {
background-color: lightgrey;
width: 300px;
border: 10px solid blueviolet;
padding: 40px;
margin: 15px;
}
</style>
</head>
<body>

<div>This text is the content inside the object. It has 40px padding (the space between the edge of the box and the content inside it), 15px margin (the invisible space around the object) and a 10px blueviolet border.</div>

</body>
</html>
